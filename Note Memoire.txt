
Demarage de projet ecommerce-cg

Instalation du projet 
1) composer create-project symfony/skeleton:"7.4.*" ecommerce-cg

2) on se place sur le dossier de travaill
cd eccommerce-cg

3) on istale les depandence naicessaire au projet que il soit complet
composer require webapp


4) Misen place de l'interface utilisateur

 php bin/console make:controller Home


5) Creation de l'entite USER
php bin/console make:user

full_name     string 255    null yes 
civility      string 255     null yes 
created_at    datetime_immutable      null no
updated_at    datetime_immutable      null yes



6) Service d'inscription 
php bin/console make   pour voir tout 

php bin/console make:registration-form  

pour la verif de email 
costincianug@gmail.com 

nom qui s'affichera est 
C.G Ecommerce 

On dit oui pour connecter l'utilisateur un fois inscription
yes 

Et on le redirige sur la page d'acceuil
app_home 15 

on instale le composant de verificaion de mail
composer require symfonycasts/verify-email-bundle


7) Créer la bdd 

php bin/console doctrine:database:create 

8) On créer la migration avec 
php bin/console make:migration

9) On envois en bdd l'entite user
php bin/console doctrine:migrations:migrate

10) Mise en forme de formulaire d'inscription 



11) On nettoy le Cache 
php bin/console cache:clear  


12) Inscription d'un utilisateur  tratiter l'envois du formulaire 


13) Faire le formulaire de connexion
php bin/console make:auth  (deprecier en sy 7.4 on utilise la paace make:security)
php bin/console make:security 


14) On créer la page  Mon compre 
php bin/console make:controller Account 


15) On crée la template qui se affiche sur toute les pages
on copie header et on renome en page_head et la templete on l'appelle aussi page_head.html.twig
et on met notre template que on a créer specificquement pour que elle se affiche sur toute les pages
